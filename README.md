# Retail Operations Intelligence: Advanced EDA + Tableau Dashboard

This is an end-to-end **business analyst portfolio project** that mimics a retail operations and commercial analytics workflow.

## Scenario
A retail operations team wants to understand profitability, return behavior, shipping delays, and customer health across channels, regions, and segments. The analyst is expected to:
1. clean messy operational data,
2. build a reusable analysis layer,
3. perform advanced EDA,
4. design a leadership dashboard in Tableau.

## What makes it "full stack"
- messy raw CSV inputs
- a cleaning + feature engineering notebook
- SQL transformations for dashboard-ready tables
- a Tableau dashboard blueprint with KPI definitions and sheet layout
- GitHub-ready documentation
